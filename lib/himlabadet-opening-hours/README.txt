This plugin add a list og locations and there opening and closing hours. Available via shortcode.

== Description ==

Add a list of locations and their opening and closing hours. Has a custom post type with rewritable permalink.
You can add a shortcode eirher in code or post content [openinghours location="" date=""]. Location is either a single
post id or "all" which lists all locations. The date parameter sets the startdate of the list.

== Installation ==

1. Upload `wp-plugin-openinghours.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the plugin settings page and set the bootstrap and tether settings as you want.
